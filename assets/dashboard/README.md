## Dashboard Story

This dashboard evaluates PYUSD from a Treasury Analyst perspective.

The analysis focuses on:
- Transaction volume and liquidity trends
- Adoption through weekly active wallets
- Entity participation via wallet categories
- Price stability relative to the $1 peg
- On-chain transaction fees and settlement reliability

Together, these metrics provide a holistic view of whether PYUSD can support
real-world treasury and payment workflows.
